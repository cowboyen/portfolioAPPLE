Overview

A 4-month collaboration project with healthcare startup Medlytic, executed by three students. The goal was to improve how healthcare professionals interact with their analytics platform, specifically for comparing data across hospitals and visualizing complex datasets. A minor part of the project was to improve the onboarding (registration process).

Key Challenges and solutions

While Medlytic's stakeholders sought comprehensive analytics capabilities, healthcare users needed intuitive, time-efficient tools. We bridged this gap by developing an optimized onboarding flow with flexible communication options, creating an intuitive dashboard for comparing healthcare metrics, and designing an interactive timeline for tracking implementation phases and pulse surveys. The solution balanced robust functionality with accessibility.

Process and methodology

The project resulted in a streamlined registration flow with enhanced privacy options, a scalable dashboard offering multiple data comparison views, and a phase-based visualization timeline. These deliverables significantly improved the platform's usability while meeting complex analytical requirements. The implemented solutions created a foundation for future platform development while satisfying both stakeholder objectives and user needs.
