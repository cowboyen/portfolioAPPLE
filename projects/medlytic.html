Overview

A 4-month collaboration project with the healthcare startup Medlytic, executed by three students. The goal was to improve how healthcare professionals interact with their analytics platform, specifically for comparing data across hospitals and visualizing complex datasets. A minor part of the project was to improve the onboarding (registration process).

Key Challenges and solutions

The project presented a key challenge: hospital leadership required efficient tools to monitor staff responses to technological and procedural changes. Our solution bridged this gap by designing an intuitive dashboard that allows hospital executives to track and compare how healthcare staff across different hospitals respond to the implementation of new technologies, AI systems, and processes. The design features comparative views between hospitals, an interactive timeline for monitoring implementation phases, and flexible data visualization options - all optimized for quick insights and strategic decision-making. This balanced approach ensures that complex data remains accessible and actionable for healthcare leadership.

Process and methodology

The project utilized both Design Sprint and Design Thinking frameworks, allowing for rapid iteration while maintaining deep user understanding. Through extensive research, prototyping, and regular stakeholder feedback sessions, we continuously refined our solutions. Through iterative testing using the Wizard of Oz method - simulating system interactions before full implementation - we gathered crucial feedback that informed design refinements and validated our solutions, ensuring they met real user needs in a healthcare management context.

Deliverables and impact

The project resulted in a streamlined registration flow with enhanced privacy options, a scalable dashboard offering multiple data comparison views, and a phase-based visualization timeline. These deliverables significantly improved the platform's usability while meeting complex analytical requirements. The implemented solutions created a foundation for future platform development while satisfying both stakeholder objectives and user needs.
